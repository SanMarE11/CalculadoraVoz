# CalculadoraVoz
#Autor: SanMarE11

Calculadora que funciona a travez de la voz

Utiliza las librerias de TalkingJavaSDK solo basta con decir un numero por numero

0:Cero

1:Uno

2:Dos

3:Tres

4:Cuatro

5:Cinco

6:Seis

7:Siete

8:Ocho

9:Nueve

.:Punto

Para realizar las operaciones

+: Sumale

-: Restale

/: Entre

x: Por

=: Igual

Y para limpiar la pantalla

C: Limpiar
